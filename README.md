# AIP Redesign Prototype

This repository is a static prototype cloned from the current AIP website at `https://aips.ab.org/`.

## Version

`v0.3` - Use plural "AB Improvement Proposals" in the site logo.

## Scope

- Home page
- All AIPs list
- AIP detail pages
- Guides index and guide detail pages
- Category, status, type, and CLA pages
- Local static assets required for the prototype

External links such as GitHub, X/Twitter, Telegram, AB Website, Developer Website, and Contact Team remain outbound links.
